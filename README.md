# gomoku_game
建立一個五子棋對戰遊戲，玩家須先建立一個帳號，登入後須先加入好友才能發起對戰，遊戲方式是判斷上、下、左、右、左斜以及右斜，也就是一個米字型，如果有相同顏色的5個棋子，即可獲勝
